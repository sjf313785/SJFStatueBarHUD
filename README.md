# SJFStatueBarHUD
简单易用的状态栏指示器框架

## 显示成功信息
'''objc
[SJFStatueBarHUD showSuccess:@"加载成功!"];
'''

## 显示失败信息
'''objc
[SJFStatueBarHUD showError:@"加载失败!"];
'''

## 显示加载信息
'''objc
[SJFStatueBarHUD showLoading:@"正在加载..."];
'''

## 显示普通信息
'''objc
[SJFStatueBarHUD showMessage:@"加载"];
'''

## 隐藏指示器
'''objc
[SJFStatueBarHUD hide];
'''
